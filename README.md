# Python Learning Spike Project

## About this project
This is my personal Python learning journey where I’m learning by actually building things instead of just reading theory.

I’m still a student and this repository is basically me trying to understand programming step by step in a structured way, while also tracking how I grow over time.

## Why I made this
I didn’t want my learning to stay random or scattered. So I’m building this space where I can actually see my progress, improve my thinking, and build small but real Python projects along the way.

## What this includes
- Python basics in my own simple understanding
- Small mini projects while learning
- My learning notes and reflections
- A step-by-step growth from beginner to more confident coding
